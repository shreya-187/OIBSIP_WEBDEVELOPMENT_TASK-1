# Booking Landing Page

A responsive flight booking landing page inspired by Booking.com, built using HTML, Tailwind CSS, and JavaScript.

## Features

- Responsive Design
- Flight Search Form
- Popular Destinations
- Special Offers
- Sticky Navigation Bar
- Modern UI
- Hover Animations

## Technologies Used

- HTML5
- CSS3
- Tailwind CSS
- JavaScript
- Font Awesome

## Folder Structure

Booking-Landing-Page/
│──index.html
├──style.css
├──script.js
└──images/

## How to Run

1. Download or Clone the repository.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension.
4. Right-click on `index.html`.
5. Select **Open with Live Server**.
